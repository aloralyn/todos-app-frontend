## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the frontend app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn storybook`

Runs the component explorer app in the development mode.<br />
Open [http://localhost:9000](http://localhost:9000) to view it in the browser.

A running version of storybook can be viewed [here](https://pm-storybook.onrender.com/?path=/story/templates--demo).
